Basic usage

Data can be loaded into viewer either by dropping files onto the application from Finder, or by opening the file from the command line. For instance, to visualize the file P306_highpass.0001, the following command can be used: open P306_highpass.0001 -a [path to application]

Once data has been loaded, one can pan through the data using horizontal scrolling on the mouse/trackpad. Areas of interest can be zoom by dragging a rectangle around it using the left mouse button. To change the magnification, of the current view, scroll left (zoom in) or right (zoom out) while holding down the command button on the keyboard. To reset zoom, double click anywhere in the view
